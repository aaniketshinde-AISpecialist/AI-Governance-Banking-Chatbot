# AI Governance Assessment — Banking Customer Support Chatbot

## Project Summary
End-to-end AI Governance assessment for a retail banking chatbot 
powered by a third-party Large Language Model (LLM).

## Governance Areas Covered
- AI Risk Assessment
- Governance Control Design
- Residual Risk Analysis
- NIST AI RMF Alignment
- Human-in-the-Loop Oversight
- Vendor Risk Governance
- Monitoring Metrics and KPIs

## Framework Alignment
- NIST AI RMF — GOVERN, MANAGE, MEASURE
- DPDP Act 2023
- RBI Guidelines
- Privacy by Design
